# Images Directory

## Required Images

Please add the following images to this directory:

1. **hero-bg.jpg** - The cosmic nebula image for the hero section background
   - Recommended size: 1920x1080 or larger
   - Format: JPG, PNG, or WebP
   - This should be the stunning cosmic nebula image provided

## Optional Images

You may also want to add:

1. **profile-photo.jpg** - Profile photo for the About section (if desired)
2. **project-*.jpg** - Screenshots for each project (A&G POS, TeaNuma, DateVoo)

The HTML and CSS are already set up to use these images - just add them to this directory with the appropriate filenames.
